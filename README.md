# ERP-MES_CNT

Система интеграции ERP (Enterprise Resource Planning) и MES (Manufacturing Execution System).

## Описание

Проект предназначен для управления производственными процессами и ресурсами предприятия.

## Возможности

- Управление производственными заказами
- Контроль исполнения производственных процессов
- Учёт ресурсов и материалов
- Мониторинг оборудования
- Отчётность и аналитика

## Установка

```bash
# Клонирование репозитория
git clone <repository-url>

# Установка зависимостей
pip install -r requirements.txt
```

## Использование

```bash
# Запуск приложения
python main.py
```

## Структура проекта

```
├── src/            # Исходный код
├── tests/          # Тесты
├── docs/           # Документация
└── config/         # Конфигурационные файлы
```
